# PowerTuner uninstaller

![](src/Resources/icon.ico)

PowerTuner uninstaller for Windows.

This is the program that runs when uninstalling from _Control panel_ or _Installed apps_ page.

This only runs if PowerTuner is installed.

## Logs
Uninstall log is saved to:
```
C:\Users\<username>\AppData\Roaming\PWTUninstall
```

_Replace **\<username\>** with your Windows account name_.
